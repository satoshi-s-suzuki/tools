How to use Click Through Demo Script

What is provided in the package?

  • There are 2 html files and 1 folder for sample demo contents.
    Sample folder contains 5 sample pictures used in sample demo.

  • ClickThroughDemoTool
    |-- index.html  : Description and Usage of the files provided
    |-- demo.html  : Demo Script File
    |-- sample        : Samle demo contents

How Can I Create a Demo?

  • 1. Download demo tool archive and unarchive
  • Download archive from the link below and unarchive to create your own demo.

    ClickThroughDemoTool.zip(47kb)
  • 2. Create a demo contents folder
  • Create a demo content folder in the same layer as demo.html.

    ClickThroughDemoTool
    |-- index.html
    |-- demo.html
    |-- sample
    |-- newdemo
  • 3. Copy screenshot files
  • Copy screenshot files to demo contents folder created in "Step1".

    DemoScript
    |-- index.html
    |-- demo.html
    |-- sample
    |-- newdemo
    |-- 1.png
    |-- 2.png
    |-- 3.png
    |-- 4.png
    |-- 5.png
    |-- XXX.png
  • 4. Open demo.html
  • Open demo.html and enter folder name(relative path) and slide counts and start Demo.



    Sample Demo Page >>

How Can I controll Demo?

  • You can move back and forth between slide with mouse buttons.

  • Right Click : Change to the previous slide
  • Left Click : Change to the next slide
